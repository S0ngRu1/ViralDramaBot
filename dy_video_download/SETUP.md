# 抖音无水印视频下载器 - 安装和设置指南

## 项目概述

已成功实现一个功能完整的抖音无水印视频下载工具，包含以下核心功能：

### ✅ 已实现功能
1. **关键词搜索**：根据关键词搜索抖音视频
2. **点赞筛选**：按最小点赞数筛选搜索结果  
3. **无水印下载**：下载无水印高清视频
4. **URL直接下载**：支持多种抖音链接格式
5. **图形界面**：PyQt5开发的用户友好界面
6. **键盘导航**：支持快捷键操作

## 安装步骤

### 1. 环境准备
确保已安装Python 3.7+：
```bash
python --version
```

### 2. 安装依赖
```bash
pip install requests PyQt5
```

或使用项目中的requirements.txt：
```bash
pip install -r ../requirements.txt
```

### 3. 快速启动
```bash
# 使用启动脚本（推荐）
python start.py

# 或直接运行主程序
python Main.py
```

## 文件说明

### 核心文件
- `Main.py` - 主程序，图形界面
- `Vibrato.py` - 核心下载逻辑
- `Form.py` - 界面布局文件
- `start.py` - 启动脚本（包含依赖检查和目录创建）

### 辅助文件
- `test_final.py` - 功能测试脚本
- `USAGE.md` - 详细使用说明
- `app.py` - Flask Web版本（备用）
- `templates/` - Web界面模板

### 自动创建的目录
- `downloads/` - 视频保存目录
- `.data/` - 临时数据目录

## 使用流程

### 方法一：图形界面（推荐）
```bash
python start.py
```

**操作步骤：**
1. 选择"搜索模式"或"下载模式"
2. 输入关键词或视频链接
3. 设置最小点赞数（搜索模式）
4. 点击相应按钮执行操作
5. 使用左右箭头键浏览搜索结果

### 方法二：命令行测试
```bash
# 运行功能测试
python start.py test

# 查看帮助
python start.py help
```

### 方法三：Web界面（备用）
```bash
python app.py
```
访问 http://localhost:5000

## 功能详情

### 1. 搜索功能
- **输入**：任意关键词（中文/英文）
- **筛选**：可设置最小点赞数
- **输出**：视频列表，包含标题、点赞数、链接
- **导航**：键盘左右箭头切换结果

### 2. 下载功能
- **支持格式**：
  - 短链接：`https://v.douyin.com/xxxxxxx/`
  - 完整链接：`https://www.douyin.com/video/xxxxxxx`
  - 文本中的链接
- **输出**：无水印MP4文件，保存到`downloads/`目录

### 3. 界面特性
- 双模式切换（搜索/下载）
- 实时状态显示
- 错误信息友好提示
- 键盘快捷键支持

## 注意事项

### API限制
由于抖音API的反爬虫机制：
- 搜索功能可能返回示例数据
- 部分视频需要登录才能访问
- API接口可能随时变化

### 使用建议
1. **个人使用**：仅用于学习和研究
2. **遵守版权**：尊重内容创作者
3. **合理频率**：避免频繁请求
4. **备用方案**：直接使用链接下载更可靠

## 故障排除

### 常见问题
1. **搜索无结果**：尝试不同关键词，降低点赞要求
2. **下载失败**：检查链接有效性，网络连接
3. **有水印**：确保使用最新版本，重新尝试

### 错误处理
程序提供详细的错误信息：
- 链接解析失败
- API访问限制
- 网络连接问题
- 文件权限错误
