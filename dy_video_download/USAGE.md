# 抖音无水印视频下载器 - 使用说明

## 快速开始

### 1. 安装依赖
```bash
pip install requests PyQt5
```

### 2. 启动程序
```bash
python Main.py
```

## 功能详解

### 一、搜索功能
1. **选择"搜索模式"**
2. **输入关键词**：如"美食"、"旅行"、"搞笑"等
3. **设置最小点赞数**：筛选高点赞视频（默认1000）
4. **点击"搜索"按钮**
5. **浏览结果**：使用左右箭头键切换搜索结果
6. **下载视频**：点击"下载"按钮下载当前显示的视频

### 二、直接下载功能
1. **选择"下载模式"**
2. **粘贴抖音链接**：支持多种格式
3. **点击"下载"按钮**
4. **查看结果**：视频将保存到`downloads/`目录

## 支持的链接格式

### 短链接格式：
```
https://v.douyin.com/xxxxxxx/
```

### 完整链接格式：
```
https://www.douyin.com/video/7316983198304111891
https://www.iesdouyin.com/share/video/7316983198304111891
```

### 包含链接的文本：
```
看看这个视频：https://v.douyin.com/xxxxxxx/ 很有意思！
```

## 操作示例

### 示例1：搜索并下载美食视频
1. 选择"搜索模式"
2. 输入"美食"
3. 设置最小点赞数为5000
4. 点击"搜索"

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| **Enter** | 执行当前模式的操作（搜索或下载） |

| **右箭头** | 下一个搜索结果 |

## 目录结构

程序运行时会自动创建以下目录：

```
downloads/
├── douyin_7316983198304111891.mp4
├── douyin_7316983198304111892.mp4
└── ...
```

视频文件名格式：`douyin_{视频ID}.mp4`

## 常见问题解决

### 问题1：搜索没有结果
**解决方法：**
- 检查网络连接
- 尝试不同的关键词
- 降低最小点赞数要求
- 直接使用视频链接下载

### 问题2：下载失败
**可能原因和解决方法：**
1. **链接失效**：检查链接是否有效
2. **需要登录**：某些视频需要抖音账号登录
3. **网络问题**：检查网络连接
4. **视频保护**：部分视频有下载限制

### 问题3：下载的视频有水印
**解决方法：**
- 确保使用最新版本
- 检查视频链接格式
- 重新尝试下载
